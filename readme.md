# Гайд по байпассу белых списков

## Регистрация в Yandex Cloud

1. Переходим на сайт: https://yandex.cloud  
2. Регистрируемся и заходим в панель управления.
3. Нажимаем **«Создать ресурс»** → **«Виртуальная машина»**.

## Резервирование IP-адреса

1. Прокручиваем страницу вниз до раздела **«Сетевые настройки»**.
2. В поле выбора сети выбираем **«Список»**.
3. Нажимаем на ссылку **«Список адресов»**.
4. Нажимаем кнопку **«Зарезервировать адрес»**.
5. Выбираем зону доступности **ru-central1-a**.
6. Получаем IP вида `51.250.xx.xx`.

> Если выпал другой IP, нажимаем на три точки рядом с адресом → **«Удалить»** и повторяем процедуру до тех пор, пока не выпадет нужный IP.

## Настройка VPS

- Конфигурацию VPS настраивайте на своё усмотрение.
- Рекомендуемый вариант:
  - **Своя конфигурация**
  - Платформа: **Intel Cascade Lake**
  - Гарантированная доля CPU: **5%**
  - Диск: **HDD 10 ГБ**

Примерная стоимость такой конфигурации — **около 730 рублей**.

## Установка панели

На VPS я устанавливал **3x-ui** и проверял работоспособность только с ней.  
С другими панелями и с «голым» Xray тестирование не проводилось.

## Настройка 3x-ui

В панели **3x-ui** создаём inbound со следующими параметрами:

- **Название**: любое  
- **Протокол**: `vless`  
- **Порт**: `443` (либо `3443` или `8443`)  
- **Безопасность**: `Reality`  

### Параметры Reality

- **SNI / TARGET** — подбираются под вашего оператора.  
- В моём случае оператор **Yota**.  
  Если у вас такой же оператор, можете попробовать:
  - **target**: `ads.x5.ru:443`
  - **sni**: `ads.x5.ru`
