# Гайд по байпассу белых списков

## Регистрация в Yandex Cloud

1. Перейдите на сайт: https://yandex.cloud  
2. Зарегистрируйтесь и войдите в панель управления.  

---

## Резервирование IP-адреса

1. Прокрутите страницу вниз до раздела **«Сетевые настройки»**.  
2. В поле выбора сети выберите **«Список»**.  
3. Нажмите на ссылку **«Список адресов»**.  
4. Нажмите кнопку **«Зарезервировать адрес»**.  
5. Выберите зону доступности **ru-central1-a**.  
6. Получите IP вида `51.250.xx.xx`.

> Если выпал другой IP — нажмите на три точки рядом с адресом → **«Удалить»** и повторите процедуру, пока не выпадет нужный IP.

## Настройка VPS

- **Своя конфигурация**
- Платформа: **Intel Cascade Lake**
- Гарантированная доля CPU: **5%**
- Диск: **HDD 10 ГБ**

Примерная стоимость — около **730 рублей**.

## Установка панели

Подключаемся к серверу:

```bash
ssh -i "путь_до_вашего_приватного_ключа" -l "логин который вы указали при этапе создания VPS" 123.123.123.123
```

Скачиваем и запускаем установку **3x-ui**:

```bash
wget https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh
chmod +x install.sh
sudo ./install.sh
```

После успешной установки выдаст такое сообщение:

```

Username: slipiz
Password: penis
Port: 1337
WebBasePath: zalupa1337
Access URL: http://ip:port/webpath
###############################################
```
