# Гайд по байпассу белых списков

## VK CLOUD
Метод полностью платный, работают (вроде) все айпишники, цена +-1200 рублей.

## Регистрация Vk Cloud
1. Переходим на [cloud.vk.com](https://cloud.vk.com)
2. Регистрация по мылу и паролю
3. Привязываем карту (привязывал виртуальную и вам так же рекомендую)
4. Подождать минут 10-15, чтобы активировались продукты

## Настройка firewall'a
1. Переходим "Виртуальные сети" -> Настройки firewall
2. Добавляем новую настройку firewall
3. Имя Группы правил - любое
4. Таблица входящего трафика -> добавить правило
5. Тип -> Все протоколы и порты, удаленный адрес -> Все IP-адреса
6. Исходящий трафик -> такие же найстройки -> выбрать любой ext-sub (главное чтобы айпи не был таким -> 10.0.0.0)


## Маршрутизаторы

1. Тип маршрутизатора -> Продвинутый
2. Название -> любое
3. Зона доступности -> Москва (MS1)
4. Сверху выбираем "Интерфейсы" -> добавить интерфейс

## Создание виртуальной машины
1. Ubuntu 24.04 -> Intel Cascade Lake (1 cpu/1gb ram)
2. Зона доступности -> Москва (MS1)
3. Сеть и настройки firewall -> Сеть -> Внешняя сеть (internet). Тут же снизу нажимаем кнопочку "Добавить группу безопасности" -> Выбираем созданный firewall -> Назначить внешний IP -> Мониторинг выключаем
4. Резервное копирование на ваш вкус

Далее инструкция такая же как и на Yandex Cloud

## Резервирование IP-адреса

1. Прокрутите страницу вниз до раздела **«Сетевые настройки»**.  
2. В поле выбора сети выберите **«Список»**.  
3. Нажмите на ссылку **«Список адресов»**.  
4. Нажмите кнопку **«Зарезервировать адрес»**.  
5. Выберите зону доступности **ru-central1-a**.  
6. Получите IP вида `51.250.xx.xx`.

> Если выпал другой IP — нажмите на три точки рядом с адресом → **«Удалить»** и повторите процедуру, пока не выпадет нужный IP.
